# Contributing

Thanks for taking a look at MiniLauncher.

## Development

```powershell
cd MiniLauncher
dotnet build
dotnet run
```

## Pull Requests

- Keep changes focused and easy to review.
- Prefer local/offline behavior by default.
- Avoid adding network dependencies unless they are optional.
- Run `dotnet build` before opening a pull request.

## Issues

When reporting a bug, include:

- Windows version
- MiniLauncher version or commit
- Steps to reproduce
- Expected behavior
- Actual behavior
