MiniLauncher portable build

Run:
  Double-click MiniLauncher.exe

Notes:
  - Alt + Space toggles the launcher.
  - Esc hides it.
  - Drag an .exe file or folder into the window to pin it.
  - You can also drag an app, shortcut, script, or folder onto Offline Store; it will be pinned automatically.
  - Use the + button to add apps and the folder button to pin folders.
  - Right-click an item to favorite, rename, open its folder, or unpin.
  - Favorite and recently opened items are sorted first.
  - Settings can enable Start with Windows, Always on top, Hide on lost focus, and Open near cursor.
  - Settings can export/import config and clean missing pinned paths.
  - Config is saved at:
    %AppData%\MiniLauncher\config.json

If Windows blocks the file:
  Right-click MiniLauncher.exe -> Properties -> Unblock -> OK.
